# 🤖 Fake Profile Detection System

An AI-based system to detect fake (bot) social media profiles using behavioral analysis and machine learning.

---

## 🚀 Features

- 📊 User input-based profile analysis
- 🤖 Bot / Real classification
- 📈 Visualization (followers, following, tweets)
- 🧠 Explanation for predictions
- 🔗 Frontend (Streamlit) + Backend (Flask)

---

## 🏗️ Tech Stack

- Python
- Streamlit (Frontend UI)
- Flask (Backend API)
- Scikit-learn (ML - upcoming)
- Pandas / NumPy

---

## ⚙️ How It Works

1. User enters profile data
2. Streamlit UI sends data to Flask API
3. Backend processes data
4. Returns prediction + confidence
5. UI displays result with visualization
